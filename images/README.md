# SR5 Dice Sprite Sheets

Place your custom die sprite sheets here. Each image must be:
- A single column with **6 faces stacked top to bottom** (face 1 at top, face 6 at bottom)
- Any square-faced size, e.g. 258 × 1548 px (258 px per face)
- `.webp`, `.png`, `.jpg`, or `.jpeg` format

The images will appear as selectable options in **Module Settings → SR5 die Sprite Sheet → Configure Image**.

The chosen sprite is registered as the **SR5 Die** system in Dice So Nice and is
available for both the Shadowrun `ds` die and the standard `d6`.
